vimrc
=====

VIm configuration package

## How to install?

    git clone git://github.com/ilblackdragon/vimrc.git ~/.vimrc_git
    cd ~/.vimrc_git
    ./install_vimrc.sh

You can choose any other directory instead of ~/.vimrc_git - this one used as an example.

## How to update?

    cd ~/.vimrc_git
    git pull --rebase
